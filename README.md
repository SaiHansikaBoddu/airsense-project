# 🌍 AirSense - Air Quality Monitoring System

## 📌 Overview
AirSense is a web-based application that provides **location-based air quality insights** using interactive maps and charts. It helps users understand pollution levels in their specific area and make better health decisions.

## 🚀 Features
- 🔍 Search location-based AQI
- 🗺️ Interactive map using Leaflet.js
- 📊 Charts using Chart.js
- 📁 Upload CSV datasets dynamically
- ☁️ Deployed on Microsoft Azure

## 🛠️ Technologies Used
- Python (Flask)
- Pandas (Data Processing)
- HTML, CSS (Frontend)
- Chart.js (Visualization)
- Leaflet.js (Map)
- Azure App Service (Deployment)

## ⚙️ How it Works
1. Upload air quality dataset
2. Data is processed using Pandas
3. AQI values are calculated
4. Results are displayed on dashboard
5. Users can explore via map and charts
